# alx-low_level_programming

## Recursion Project

This project includes the following tasks:

	- Writing a function that prints a string
	- Writing a function that reverses a string 
	- Writing a function returns the length of a string
	- Writing a function that returns the factorial of a given number.
	- Writing a function that returns the value of x raised to y.
	- Writing a function that returns natural square root of a number.
	- Writing a function that checks if a number is prime or not.
	- Writing a function that checks if a string is palindrome or not.

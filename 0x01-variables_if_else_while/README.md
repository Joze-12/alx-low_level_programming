# alx-low_level_programming

__0x01. C - Variables, if, else, while__

This Project includes the following exercises:

	* __Positive anything is better than negative nothing__ Prints whether the number stored in the variable n is positive or negative.
	* __The last digit__ Prints the last digit of the number stored in the variable n.
	* __I sometimes suffer from insomnia. And when I can't fall asleep, I play what I call the alphabet game__ Prints the alphabet in lowercase.
	* __alphABET__ Prints the alphabet in lowercase and then in uppercase
	* __When I was having that alphabet soup, I never thought that it would pay off__ Prints the alphabet in lowercase.
	* __Numbers__ Prints all single digit numbers of base 10 starting from 0.
	* __Numberz__ Prints all single digit number os base 10 starting from 0.
	* __Smile in the mirror__ Prints the lowercase alphabet in reverse.
	* __Hexadecimal__ Prints all the numbers of base 16 in lowercase.
	* __Patience, persistence and perspiration make an unbeatable combination for success__ prints all possible combinations of single digit numbers.
	* __00...99__ Prints the numbers from 00 to 99

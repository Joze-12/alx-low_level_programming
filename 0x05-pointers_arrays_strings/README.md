# C - Pointers, arrays and strings

This project contains the following tasks:

	- writing a function that takes a pointer to an int as parameter and updates the value it points to to 98.
	- writing a function that swaps the valuews of two integers.
	- writing a function that returns the length of a string
	- writing a function that prints a string
	- writing a function that prints a string in reverse
	- writing a function that reverses a string
	- writing a function that prints every other character of a string
	- writing a function that prints half of a string
	- writing a function that prints n elements of an array of integers.
	- writing a function that copies the string pointed by 'src', including the terminating null byte(\0), to the buffer pointed to by 'dest'

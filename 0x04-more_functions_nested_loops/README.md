# alx-low_level_programming

## C - More functions, more nested loops

This Project contains taskes related to:

	* Creating a function that checks for uppercase character
	* Creating a function that checks for a digit
	* Creating a function that multiplies two integers
	* Creating a function that prints the numbers from 0 - 9
	* Creating a function that prints the numbers from 0 - 9 except 2 and 4
	* Creating a functions that prints 10 times the numbers from 0 - 14
	* Creating a function that draws a straight line in the terminal
	* Creating a function that draws a diagonal line on the terminal
	* Creating a function that prints a square
	* Creating a function that prints the numbers from 1 - 100
	* Creating a function that prints a triangle
